# 🛡️ نظام حماية ندى (Security-Pro-Bot)

مشروع برمجي بلغة **Python** مخصص لفحص الروابط المشبوهة وعزلها لحماية المستخدمين من الاحتيال الإلكتروني، خاصة روابط النتائج الوهمية.

## ✨ المميزات:
- **التمييز الذكي:** يفرق بين المواقع الرسمية (مثل وزارة التربية) والروابط المزيفة.
- **عزل التهديدات:** يحول الروابط الخطيرة إلى صيغة نصية غير قابلة للنقر.
- **سهولة الاستخدام:** يعمل مباشرة عبر بيئة **Termux**.

## 🚀 طريقة الاستخدام:
شغل الملف باستخدام الأمر:
`python security_pro.py`
